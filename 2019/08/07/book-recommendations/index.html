推荐书籍

发表于 2019-08-07 更新于 2020-01-16 分类于 书痴

一直以来读过的书中有印象的推荐（记忆力不行，慢慢补）

推荐

《百年孤独》——加西亚·马尔克斯（坚持看下去就不容易了）

《小王子》（不用说了，极力推荐）

《乌合之众：大众心理研究》——古斯塔夫·勒庞

《夜航》——圣埃克苏佩里

《奇迹男孩》——R.J.帕拉西奥

《毛泽东传》——迪克·威尔逊

《失控》《必然》《科技想要什么》——KK大叔

《半小时漫画系列》——陈磊（好像全都看了，讲的还不错，挺系统的，看的快）

《边城》——沈从文

《苏东坡传》——林语堂

《人类·今日·未来简史》——尤瓦尔·赫拉利

《论中国》——亨利·基辛格

《迷人的材料》

《心经》

《约翰·克里斯朵夫》–罗曼罗兰 从此变多了个朋友，他的名字叫约翰克里斯朵夫

《》

（专业书貌似没认真看完的，我的错，不过有可多想推荐的，可以私信）
