斐波那契数列

发表于 2020-03-03 分类于 剑指Offer

大家都知道斐波那契数列，现在要求输入一个整数n，请你输出斐波那契数列的第n项（从0开始，第0项为0）。n <= 39

解法

递归

斐波那契数列标准公式为：

$$
F(1)=1, F(2)=1, F(n) = F(n-1)+F(n-2) ( n >= 3, n \in N^*)
$$
